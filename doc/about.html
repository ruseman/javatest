JTest

About

This simple Java application tests the user on a series of questions specified in a certain file stored online.

Technologies

This application relies on several 3rd party software technologies, including but not limited to

  • Gradle, a buildscript system
  • Ant, a scripting system, implemented in Gradle
  • Guava, a Google utility library
  • JSON, JavaScript Objective Notation
  • GSON, a Google implementation of a JSON interpreter for Java
  • VoxioLib, a WIP Java utility library authored by me

The GUI is rendered using the Java Abstract Window Toolkit and Swing Toolkit.

Mechanics

The game itself is simple enough. The player begins with 10 points. A correct answer rewards the player with one point, and an incorrect answer costs one point. If the player runs out of points, then they've lost. If they make it through all of the questions with a positive number of points, they've won. There's not a whole lot more to it than that.
